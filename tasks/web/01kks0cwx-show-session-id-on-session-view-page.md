# Show session ID on session view page

## Objective

Display the session ID on the session view page so users can quickly identify and reference the current session.

## Tasks

- [ ] Identify where the session ID is available in the session data model
- [ ] Add the session ID display to the session view page header/metadata area
- [ ] Ensure the session ID is selectable/copyable

## Acceptance Criteria

- The session view page displays the session ID visibly
- The session ID is shown without requiring extra clicks or navigation
- The displayed ID matches the actual session identifier
